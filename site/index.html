Two state-of-the-art neural nets run in parallel; the best output wins for each stem. Takes wav, mp3, flac, m4a. ~5 minutes for a 4-minute song.

How It Works

  1. Download the DMG, drag Stem Lab to Applications.
  2. Open it once via "Open Anyway" — first run installs the audio engine.
  3. Drop any song on the window or the icon.
  4. Hit Open Stems Folder. Four stems, next to your song.

Private by Design

Fully offline. Both models run on your GPU via Metal. No uploads, no processing queue, no account, no telemetry — the app never touches the network.

Built as the free alternative to cloud stem splitters. The entire pipeline is open source — read it, build it yourself, or grab the DMG above.
